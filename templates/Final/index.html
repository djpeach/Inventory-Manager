Dashboard for Inventory Management

Made for Sutton Garten Co., for CS 230

Sales

The Sales Team enters new customers and adds orders to the system.

This is intended for the front desk, salespeople, and others who need to create order tickets.

Dashboard

The Dashboard displays information about tubs and bottles, including status and count.

This is for everyone to view the status of inventory, but specifically for management.

Warehouse

The Warehouse is responsible for filling, updating, and shipping bottles and tubs.

This is intended for warehouse workers, but is also used by others who need to handle shipping.
